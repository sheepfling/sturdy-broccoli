# Shim Route Traces

This directory stores the generated per-route trace packets used by the
standard-binding route-parity lane.

Current contents:

- `java-standard-2025-{jpype,py4j}.json`: Java 2025 runtime-capability traces
- `cpp-standard-2010-{grpc,pybind}.json`: C++ 2010 core-exchange traces
- `cpp-standard-2025-{grpc,pybind}.json`: C++ 2025 runtime-capability traces
- `summary.json`: route-trace rollup
