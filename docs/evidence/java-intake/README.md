# Java Intake Evidence

This family stores generated Java intake and certification packets for the
2010 and 2025 standard-shim routes over JPype and Py4J.

Canonical order:

1. 2010 intake packet
2. 2025 intake packet

Current contents:

- `java-2010-standard-shim-2010-{jpype,py4j}.{json,md}`: generated 2010 Java
  standard-shim certification packets
- `java-2025-standard-shim-2025-{jpype,py4j}.{json,md}`: generated 2025 Java
  standard-shim certification packets
