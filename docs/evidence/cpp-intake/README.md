# C++ Intake Evidence

This family stores generated C++ intake and capsule-certification packets for
the standard-route and external-profile probes.

Canonical order:

1. standard-route intake packet
2. external-profile intake packet

Current contents:

- `cpp-standard-2010-2010-{grpc,pybind}.{json,md}`: generated intake packets
  for repo-owned 2010 standard-route C++ SDK probes
- `cpp-standard-2025-2025-{grpc,pybind}.{json,md}`: generated intake packets
  for repo-owned 2025 standard-route C++ SDK probes
- `cpp-standard-2010-external-profile-2010-{grpc,pybind}.{json,md}`:
  retained external-profile adapter-smoke packets
