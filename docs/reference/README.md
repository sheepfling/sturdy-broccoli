# Reference

This directory holds acquisition notes and retained archive copies for the HLA 2010 reference material.

Canonical working source material for this repo lives under:

- `../../specs/ieee-1516-2010/`

That directory is the canonical unpacked standards/source bundle because the code
and analysis tools already reference it directly.

Retained in this directory:

- `hla-2010-specs.zip`
  the original PDF-only archive imported from `INBOX`

The ZIP is intentionally not unpacked here. The PDFs already exist in
`../../specs/ieee-1516-2010/`, and maintaining a second extracted copy would
create unnecessary duplication.
