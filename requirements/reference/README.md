# Requirements Reference

This directory contains the broader clause catalogs, detailed reconciliation
tables, and packet-facing harmonization artifacts.

These files are not the normal human edit surface for repo maintenance.

For the ordinary active traceability lane, go back to:

1. [`../active_service_rows.csv`](../active_service_rows.csv)
2. [`../../docs/requirements_trace_one_method.md`](../../docs/requirements_trace_one_method.md)
3. [`../../docs/requirements_edit_one_row.md`](../../docs/requirements_edit_one_row.md)
4. [`../surface_manifest.json`](../surface_manifest.json)

These reference catalogs primarily reconcile the IEEE 1516-2010 (2010 edition),
IEEE 1516.1-2010 (2010 edition), and IEEE 1516.2-2010 (2010 edition) editions against the repo's current
implementation and verification surfaces.

Use this directory only when you are doing broader work such as:

- clause-by-clause reconciliation
- imported packet harmonization
- family-level bridge maintenance
- framework or OMT backfill
