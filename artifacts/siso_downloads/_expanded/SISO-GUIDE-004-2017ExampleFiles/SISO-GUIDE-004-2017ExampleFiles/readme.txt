This SISO-GUIDE-004-2017ExampleFiles folder contains the following:
- cbml-example-expressions folder: contains specific-use example schemas and XML files discussed in the Guide
- this readme.txt file

NOTE: To work with the example files, download and unzip the C-BML standard schema files from the SISO web site
(SISO Product Data Files): https://www.sisostds.org/Schemas.aspx

The example files use a relative addressing scheme to refer to the XML schemas from the standard.
Just copy the folder named "Schemas" from the C-BML download archive into the Guide folder 
"SISO-GUIDE-004-2017ExampleFiles" (where this readme file is located). When you open any of the example
schemas and XML documents, the relative addresses will resolve correctly to allow direct validation 
of the example files against the example schemas and against the XML schemas from the C-BML standard.
